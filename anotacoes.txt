# Configurando Samba

No terminal da VM, efetuar atualizações:
> sudo apt-get update

Instalar samba
> sudo apt-get install samba

Directorio samba
/etc/samba/

Criar arquivo de bkp das configurações do samba
> mv smb.conf smb.conf.bkp

Criar novo arquivo smb
> nano smb.conf

conteudo
-------
[global]
workgroup = filipe

[VM]
comment = Pasta Compartilhada
path = /mnt
valid users = share
writable = yes
-------

Dar permissao na mnt
> sudo chmod 777 /mnt

Verificar usuarios ativos
> sudo pdbedit -L

Criar usuario share
> sudo useradd share

Adicionar usuario no samba
> smbpasswd -a share

Restartar serviço
> sudo systemctl restart smbd.servic

No windows, habilitar recurso
Painel controle > Programas e Recursos> Ativar ou desabilitar recursos do windows
Ativar Suporte para Compartilhamento de arquivos SMB 1.0/CIFS
